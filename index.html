Tutoriel OpenVAS + Metasploitable + Juice Shop (Windows)

OBJECTIF

VM OpenVAS (scanner)
VM Metasploitable
Juice Shop (Docker)
Tout dans le même réseau

ETAPE 1 - Installer VirtualBox

Télécharger et installer VirtualBox (Next → Next → Install)

ETAPE 2 - Installer Docker + WSL

Activer WSL

wsl --install

Explication : wsl = Linux dans Windows --install = installe automatiquement Ubuntu

Vérifier Docker

docker --version

ETAPE 3 - Créer VM OpenVAS

RAM : 6 Go
CPU : 2-4
Disque : 40 Go
Réseau : Host-Only Adapter

ETAPE 4 - Installer OpenVAS

Mise à jour

sudo apt update && sudo apt upgrade -y

sudo = admin
apt update = liste logiciels
apt upgrade = mise à jour
-y = accepter automatiquement

Installation

sudo apt install gvm -y

Initialisation

sudo gvm-setup

Télécharge vulnérabilités
Configure PostgreSQL
Crée admin

Vérification

sudo gvm-check-setup

Démarrer

sudo gvm-start

IP VM

ip a

Accès navigateur :

https://IP_VM:9392

ETAPE 5 - Metasploitable

Importer fichier .OVA dans VirtualBox
Configurer réseau Host-Only
Login : msfadmin / msfadmin

IP

ifconfig

ETAPE 6 - Juice Shop

docker run -d -p 3000:3000 bkimminich/juice-shop

-d = arrière-plan
-p = port
image = Juice Shop 

http://localhost:3000

ETAPE 7 - Test réseau

ping IP_Metasploitable

Créer Target

Lancer Scan
